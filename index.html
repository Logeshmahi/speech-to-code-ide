Voice command cheat sheet

Structure

  • "define function name with parameters a and b"
  • "class name inherits from base"
  • "if / elif / else / while / for x in y"
  • "try / except error / finally"
  • "raise error" — knows value error, type error, key error, etc.
  • "with expr as var"

Statements

  • "print expr" · "return expr"
  • "var equals expr"
  • "import module" · "from module import name"
  • "comment text" · "pass" · "break" · "continue"

Symbols (spoken → typed)

  • plus, minus, times, divided by, modulo, power
  • equals equals, not equal to, greater/less than (or equal to)
  • open/close paren · bracket · brace, colon, comma, dot, quote
  • numbers: "twenty five" → 25

Control

  • "new line" · "indent" · "dedent"
  • "delete line" · "clear all"
  • "save file" · "run code"
  • "switch to dictation mode" / "switch to code mode"
